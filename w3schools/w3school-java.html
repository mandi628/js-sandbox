External scripts are practical when the same code is used in many different web pages. They have the file extension .js. To use an external script, put the name of the script file in the src attribute of a script /script tag.

Advantages of using external scripts include:

  • It separates HTML and code
  • It makes HTML and JavaScript easier to read and maintain
  • Cached JavaScript files can speed up page loads

The add several script files to one page - use several script tags:

  • script src="myScript1.js" /script
  • script src="myScript2.js" /script

You can reference an external file source with a full URL, a file path (ex. in a different folder), or without a path (in the same folder as the HTML file).
