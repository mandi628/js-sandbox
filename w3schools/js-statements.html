JavaScript Statements

A computer program is a list of "instructions" to be "executed" by a computer.

These programming instructions are called statements.

A JavaScript program (or any language) is a list of programming statements.

In HTML, JavaScript programs are executed by the web browser.

JavaScript statements are made of: values, operators, expressions, keywords, and comments.

document.getElementById("demo").innerHTML = "Hello Dolly.";

This statement tells the browser to write "Hello Dolly." inside an HTML element with id="demo".

Most programs have many statements. They are executed, one by one, in the same order they are written.

Semicolons (;) are used to separate statements in JavaScript. Each language uses a different cue as statement separators. (Python, for example, does not require a separator. CSS also uses semicolons.)

  • let a, b, c; // Declare 3 variables
  • a = 5; // Assign the value 5 to a
  • b = 6; // Assign the value of 6 to b
  • c = a + b; // Assign the sum of a and b to c.

When separated by semicolons, multiple statements on one line are allowed: a = 5; b = 6; c = a + b;

JavaScript ignores multiple spaces. You can add white space to your script to make it more readable. (Python also allows while space.) let person = "Hege"; is the same as let person=;. It is a good practice to put spaces around operators.

For readability, programmers often like to avoid code lines longer than 80 characters. If it doesn't fit on one line, a good place to break it is after an operator.

Statements can be grouped together into code blocks, inside curly brackets {...}. Code blocks defines statements that should be executed together. You'll see these inside functions. Standard indentation for code blocks is two spaces. (Python best practice is four, but there is no requirement.)

Statements often start with a keyword to identify the action to be done. The Reserved Words Reference lists all JavaScript keywords. The keywords used in the tutorial are here:

Keyword Description
var Declares a variable
let Declares a block variable
const Declares a block constant
if Marks a block of statements to be executed on a condition
switch Marks a block of statements to be executed in different cases
for Marks a block of statements to be executed in a loop
function Declares a function
return Exits a function
try Implements error handling to a block of statements
