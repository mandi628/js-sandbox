JavaScript Display Possibilities

Data can be "displayed" by JS by writing into an HTML element with innerHTML; writing into the HTML ouput with document.write(), writing into an alert box with window.alert(), or writing into the browser console with console.log().

Using innerHTML

To reference the HTML element you want to change, use the document.getElementById(id) method, as demonstrated on the Intro page. The innerHTML property defines the content written to the page.

Example:

  • <p>My First Paragraph</p>
  • <p id="demo"></p>
  • <script>
  •   document.getElementById("demo").innerHTML = 5 + 6;
  • </script>

And now, in action...

Using document.write()

This method is most commonly used for testing purposes. Using document.write() after a page loads will delete all existing HTML.

Example:

  • <script>
  •   document.write(5 + 6);
  • </script>

Now, to see it in action... WARNING: It will overwrite the whole document if you use it after the page has finished loading.

Using window.alert()

This code will display an alert box. The "window" keyword is optional.

  • <script>
  • window.alert(5 + 6);
  • </script>

And now the live example... Leaving the code as-is makes the pop-up happen when the page loads.

Putting it in a button lets me control when to call it.

Using console.log()

The console.log() is most often used for debugging purposes. The console is a developer feature (available in most browsers, I'm sure, but specifically in Chrome). You have to have the console open to be able to read the output.

  • <script>
  • console.log(5 + 6);
  • </script>
